# sam3.cpp

Local packaged `sam3.cpp` runtime files for `ofxGgmlSam`.

This folder follows the openFrameworks addon-library layout:

- `include/` for runtime headers copied by `scripts/install-sam3-cpp.*`
- `src/` for runtime sources copied by `scripts/install-sam3-cpp.*`
- `source/` for the ignored upstream checkout used during local setup and builds
- `lib/` for ignored local build outputs when available

Downloaded runtime files are local-only and should not be committed.
