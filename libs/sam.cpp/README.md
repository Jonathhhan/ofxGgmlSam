# sam.cpp

Local packaged `sam.cpp` runtime files for `ofxGgmlSam`.

This folder follows the openFrameworks addon-library layout:

- `include/` for runtime headers copied by `scripts/install-sam-cpp.*`
- `src/` for runtime sources copied by `scripts/install-sam-cpp.*`
- `source/` for the ignored upstream checkout used during local setup
- `lib/` for ignored local build outputs when available

Downloaded runtime files are local-only and should not be committed.
