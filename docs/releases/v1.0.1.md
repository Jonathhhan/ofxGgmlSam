# ofxGgmlSam v1.0.1

## Compatibility

- openFrameworks: `of_v20260406_vs_64_release`
- minimum `ofxGgmlCore`: `1.0.1`
- `OFXGGML_SAM_VERSION_STRING`: `1.0.1`

## Highlights

- Added independent SAM addon version metadata.
- Exposed version metadata through `ofxGgmlSam.h`.
- Added validation coverage for public version metadata.
- Kept SAM-specific code out of `ofxGgmlCore`.
- Preserved the root-level `ofxGgmlSamPointExample` as the first focused
  example lane.

## Runtime Scope

This release is a bridge and skeleton release. It includes public request/result
types, prompt validation helpers, a replaceable bridge backend, headless tests,
and a point example skeleton.

It does not include a complete SAM/SAM2/SAM3 model runtime adapter yet.
